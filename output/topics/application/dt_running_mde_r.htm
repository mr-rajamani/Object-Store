Getting Started with Altair Object Store
  • About Altair Object Store

    Object Store is an enhanced next-generation platform core that has combined capabilities of numerous services.

  • Object Store Features

    Altair Object Store is an advanced content management system that combines the capabilities of numerous services and an action execution engine that applies Access Control List (ACL) and other authorization policies for enhanced security.

  • Benefits of Object Store

    The Altair Object Store allows you to model engineering objects that are independent of the user interface and can be consumed in a variety of applications via APIs.

  • Altair Object Store Platform Architecture

    The Object Store architecture includes, in addition to the central Object Store engine that manages configuration and execution, an environment manager, a multi-model database, an event bus, and gateways. Investigate the many different parts that make up the Object Store architecture.

    This section provides the prerequisites, instructions to install Object Store, verification of Object Store installation, and Object Store service commands.

Running MDE

The following code is used for running MDE:
from altair.mde import GeneratePreview
from altair.system import Fs, Content
from altair.system import ActionExecOptions, WaitForActionCompletion

# Specific where to run the MDE i.e on which Appliance
execOptions = ActionExecOptions(execEnvFilter={"path": "$.name", "value": "Sessions"})

# Execute Preview Gen MDE for the given model file
mdeAction = await GeneratePreview(inputFile=modelFile.getRef(), session=session, execOptions=execOptions)
