Create a Relation for the Content Definition

Author the content definition by creating a relation.

Perform the following steps to create a relation for the content definition.

  1. Click next to the Relations.
    The Create Relation is displayed.
  2. Enter the Relation name.
  3. Select the Content type of related content from the drop-down list.
  4. Select the From cardinality [from this content to related content] from the drop-down list.
  5. Select the To cardinality [from this content to related content] from the drop-down list.
  6. Click Create.
    The content definition lists the created relation.
